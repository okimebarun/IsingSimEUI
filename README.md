# IsingSimEUI.jl
Interactive Ising Model Simulator with Julia and Electron.
+ Requirements: Julia/REPL, Electron.jl

# Usage
+ Run the following:
```console
shell> julia IsingSimEUI.jl
```
+ Or, run as a package:
```console
julia> cd("IsingSimEUI")
pkg> activate .
julia> using IsingSimEUI
julia> IsingSimEUI.winmain(1)
```
